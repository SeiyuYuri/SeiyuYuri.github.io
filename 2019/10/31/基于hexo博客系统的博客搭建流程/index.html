应某位巨佬的委托，介绍一下基于hexo博客系统的博客搭建流程

步骤有少许繁琐，请多多谅解。

在这篇文章中，你会：

  1. 下载并安装 "Git分布式版本控制系统",以及针对博客的基础操作
  2. 下载并安装 "Nodejs"以搭建运行环境
  3. 下载并安装 "hexo系统"开始你的博客之旅!!

1.下载 Git

然后请安装(next就对了)

你的程序列表之中应该出现了Git Bash,这是我们之后要用到的.

2.下载Nodejs

然后请安装(同样next就对了)

这是运行环境,不会直接使用.

3.安装hexo系统

前期准备:

请准备好一个文件夹,以存储hexo相关文件.下文中的指令若未特别说明将默认在此目录下进行.(默认: D://hexo )

3.1. 打开Git Bash,直接输入安装指令 (注:显示 $ 即可输入指令.)

电脑将自动下载hexo文件至nodejs文件库.

3.2.将hexo安装至指定文件夹(之前准备的文件夹)

进入文件夹 安装hexo相关文件

hexo会自动下载一些文件到这个目录，包括node_modules，这里有所有已安装插件

##到此处,hexo系统已经搭建完成.
